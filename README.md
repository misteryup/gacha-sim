# 🎲 SealM on Cross Gacha Simulator

Simple web-based gacha simulator for SealM on Cross inspired by the actual published rates from the game.

This project exists for fun, curiosity, and community awareness — not to attack the game or spread negativity.  
The simulator uses publicly available gacha probabilities published by the Game Master / official event notices.

The goal is simple:

> Help players understand probability, RNG (Random Number Generatio), and how brutal gacha can statistically be 😭

---

## ✨ Features

- Simulate opening any amount of gacha boxes
- Uses published drop rates from the game
- Aggregated results (duplicate items merged automatically)
- Jackpot highlight for Mythic / rare items
- Lightweight HTML + JavaScript only
- Easy to update for future gacha events

---

## ⚠ Disclaimer

This simulator:

- does **NOT** guarantee real in-game results
- does **NOT** predict your account luck
- does **NOT** accuse the game of fraud/scam
- is purely a probability simulation based on published rates

Actual in-game outcomes may vary due to:
- RNG implementation
- server-side mechanics
- hidden pity systems (if any)
- event modifiers
- your cursed account 🤣

---

## 🧠 Why This Exists

Many players open gacha emotionally:
- “1 more pull”
- “surely next one”
- “I already spent too much”

But statistically:
- low-rate items are REALLY low
- RNG has no memory
- opening more boxes does not guarantee success (unless pity exists)

This simulator is meant to encourage:
- healthy spending habits
- realistic expectations
- smarter gaming decisions
- less rage quitting 😆


Then open:

```bash

```
